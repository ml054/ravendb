This wizard is designed to assist you in RavenDB server setup. It will guide you through step-by-step instructions on how to set up your RavenDB instance and how to make it secure.

First, please choose configuration mode:

- RavenDB will be running using https protocol but certificates will be automatically generated using Let's Encrypt service.

- RavenDB will be running using https protocol with certificates provided by user.

- RavenDB will be running using http protocol. Everyone who can access server will have admin privileges. This option is useful for development.
